# Sistema de Agendamento

## Descrição

Sistema de Agendamento desenvolvido com HTML, CSS e JavaScript. A aplicação permite cadastrar agendamentos informando nome, data e horário, exibindo os registros diretamente na página após a validação dos dados.

## Objetivo

Praticar conceitos fundamentais de desenvolvimento web, incluindo manipulação do DOM, eventos, validação de formulários e criação dinâmica de elementos utilizando JavaScript.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Cadastro de agendamentos.
- Validação de campos obrigatórios.
- Validação do campo nome, permitindo apenas letras e espaços.
- Exibição dos agendamentos na tela.
- Limpeza automática dos campos após cada cadastro.

## Conceitos Praticados

- Estruturação de páginas com HTML.
- Estilização utilizando CSS.
- Manipulação do DOM.
- Eventos de clique.
- Validação de formulários.
- Expressões regulares (Regex).
- Criação dinâmica de elementos HTML.

## Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/sistema-agendamento.git
```

2. Acesse a pasta do projeto.

3. Abra o arquivo `index.html` em qualquer navegador.

## Como Utilizar

1. Digite o nome.
2. Selecione uma data.
3. Escolha um horário.
4. Clique no botão **Agendar**.
5. O agendamento será exibido abaixo do formulário.
